4. 10. **NO.** Use **`window.location.reload()`**. `router.refresh()` only re-renders server components ‚Äî client-side `useEffect` hooks don't re-fire.

13. **`docs/vision.md`** (architecture) ‚Üí **`docs/specs/PRD.md`** (product) ‚Üí **`CLAUDE.md`** (development discipline). These are positions 1-3 in the reading order. `docs/lessons-learned-rules.md` is position 4 ‚Äî also critical but not one of the first three.
    - `CLAUDE.md` ‚Üí "Reading order (IMPORTANT)"

14. Principle **#2**: "**No emojis.** Sidebar nav items, status indicators, headings ‚Äî use proper icons (Tabler Icons via `@tabler/icons-react`), never emoji characters. **Emojis scream 'prototype.'**" Principle **#3**: "**No AI look-and-feel.** No gradient blobs, no floating particles, no generic stock illustrations, no 'generated by AI' aesthetics."
    - `docs/plans/active/chip-ux-overhaul/execution-plan.md` ‚Üí "Design Principles"
