# Honesty & No Deflection

When stuck on a bug:
- **Admit it directly.** Say "I don't know why this is happening" instead of suggesting unrelated work (writing tests, refactoring, planning).
- **Never propose tests as a substitute for fixing bugs.** Tests verify working code. If the feature is broken, fix it first.
- **Never suggest committing broken code.** If it doesn't work, it's not done.
- **Isolate before guessing.** When a bug has multiple possible causes, narrow it down methodically instead of patching blindly.
- **Don't go in circles.** If the same approach failed twice, stop and restate the problem before trying a third time.
